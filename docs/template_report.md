# templates/report.html

## Overview
The **analysis result report page** for a single UBX file. Auto-reloads while
conversion is in progress and, once complete, shows metadata plus a stack of
Chart.js time-series sections. Visual structure mirrors
`compare4_report.html` (sticky header, numbered section titles, cards).

---

## URL
`GET /report/{rid}`

---

## Jinja2 Template Variables

| Variable | Type | Description |
|---|---|---|
| `r` | dict | DB results row (id, filename, uploaded_at, epoch_total, epoch_missing, kmz_path, status, error, ...) |
| `graph_data` | dict or None | Graph JSON generated by ubx2kmz (`_graph.json`) |

---

## Screen Layout

### Sticky Header
- `ubX-ray` brand mark, thin separator, `📊 Analysis Report` label
- Right-aligned `.hbtn` buttons:
  - `⬇ KMZ` (when `kmz_path` exists) -> `/download?path={kmz_path}`
  - `🗺 Map` (when `kmz_path` exists) -> `/map/{rid}` (opens in a new tab)
  - `🕘 Recent` -> `/recent`

### ① Metadata
Single card containing:
- Status badge (`done` / `queued` / `running` / `error` -- error message is
  appended after an em dash)
- Two-column `meta-table`: Filename, Upload Time (UTC), Total Epochs,
  Missing Epochs
- Action row:
  - `⬇ Download KMZ` (primary) + `🗺 View on Map` (opens in new tab) when
    conversion is done
  - Otherwise a processing or error status badge

### ② Position Accuracy (2D vs 3D)
Chart: `#accChart`. Overlaid 2D (blue) / 3D (orange) lines vs iTOW.

### ③ Fix Type Status
Chart: `#fixChart`. Stepped line 0..5 (No fix / DR / 2D / 3D / GNSS+DR / Time).
Rendered only when `graph_data.fix_type` is present.

### ④ Satellite Signal Density (CN0)
Chart: `#cnoChart`. Single green line = Top-5 average CN0 across satellites
with `qualityInd >= 4`. The per-satellite scatter dataset was removed for
performance. Rendered only when `cno_top_avg` has length > 0.

### ⑤ Jamming & Spoofing Status (UBX-SEC-SIG)
Chart: `#secChart`. Two stepped lines: jamming (red, 0..2) and spoofing
(amber, 0..3). Rendered only when `sec_labels` is non-empty.

## Per-Chart Controls
Every chart card has a `.chart-controls` row with `.ctl-btn` buttons:
- `🔍 Reset All Zooms` -- `resetAllZooms()` restores original x range on
  every instantiated chart (acc / fix / cno / sec)
- `👁 Hide Chart` -- `toggleChart(boxId, btn)` toggles `.chart-wrap` display

Chart interaction:
- `chartjs-plugin-zoom` + `hammerjs` for wheel zoom / drag pan on the x axis
- `syncCharts(sourceChart)` snaps min/max to a 100 ms grid and propagates the
  range to the other three charts so all time axes stay aligned
- `formatITOW(itow)` converts GPS iTOW to `HH:MM:SS` UTC for tick labels and
  tooltip titles
- Shared `commonScalesX`, `commonTooltip`, `zoomOptions`, and
  `FIXED_Y_WIDTH = 85` keep left-hand y axes aligned across charts

## Auto-Reload Logic

```html
{% if r.status != 'done' %}
<script>setTimeout(() => location.reload(), 3000);</script>
{% endif %}
```

Full-page reload every 3 s while status is `queued`, `running`, or `error`.
Once `done`, the reload stops.
